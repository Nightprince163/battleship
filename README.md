\## 概述

基于node的网页版棋盘类海战游戏，取名为Battleship。游戏在一个16\*16的棋盘中进行，回合制，双方轮流行动。该游戏需要开两个网页，可以两台电脑对战，也可以在一台电脑上开两个网页轮流切换。

\## 游戏规则

\- 棋子尺寸：航空母舰1\*4（格，下同），战列舰1\*4，驱逐舰1\*3，导弹艇1\*2，战斗艇1\*1。

\- 游戏方分为红方和蓝方，双方开局各拥有航空母舰1，战列舰2，驱逐舰2，导弹艇2，战斗艇2。可以在server.js中自定义棋子数量。

\- 每盘游戏初始化时，棋盘会随机生成5~16个格子的障碍（小岛），在中间区域随机分布。开局前，先由红方布局，选择己方各艘船的位置和朝向，但是只能在棋盘中垂线的左侧。红方布局完成后，由蓝方布局，只能在棋盘右侧。

\- 双方布局完成后，轮流开始。在一回合内，每艘船可以选择移动或者攻击或者转向，但每艘船每回合只能做一项动作。

\- 每艘船的移动速度是1格，战斗艇可以横向或纵向上下左右移动，而其他船只能沿摆放方向前后移动1格，或者选择转向。

\- 航空母舰的攻击距离是5格，即对方船任意部位距离航空母舰任意部位的横向或者纵向距离为小于5格，即可受到攻击。战斗艇的攻击距离是3格。其他所有船只攻击距离均为4格。航空母舰的攻击可以跨越障碍（小岛或者其他船只，包括己方和对方的船），但其他船的攻击不能跨越障碍。

\- 每艘船的血量等同于其长度（格数）。攻击时，在界面自动投一颗六面骰子，结果为1时，攻击力为1；结果为6时，攻击力为2；其他结果视为miss。对方船只扣除攻击力相应血量，血量为0时被击沉。

\- 一方所有船只均操作完毕后，回合结束，切换另一方回合，由另一方操作鼠标。一方所有船只都被击沉视为战败。
